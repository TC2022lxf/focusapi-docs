FocusAPI 公有云按 **额度** 扣费。不同模型的价格由 **模型倍率**、**补全倍率** 等参数决定，控制台 **模型广场** 会展示参考价格。

## 核心概念

| 术语 | 说明 |
| --- | --- |
| **额度** | 账户可用余额；不足时请求可能失败 |
| **模型倍率** | 不同模型相对基础价格的乘数 |
| **补全倍率** | 输出 token 相对输入的额外计费因子 |
| **按量计费** | 按 prompt / completion token 用量计费（常见） |
| **按次计费** | 部分能力按每次调用固定扣费 |

术语与控制台文案保持一致，详见项目术语表（内部维护）。

## 如何查看用量

1. 登录控制台
2. 打开 **日志** 或 **用量 / 统计** 相关页面（以实际菜单为准）
3. 单次文本请求的 `usage` 字段（`prompt_tokens`、`completion_tokens`）可作为参考，见 [文本对话](/models/chat)

## 充值

- 在控制台 **充值** 页面完成支付或兑换
- **兑换码** 可增加额度

<Note>
  价格与活动可能调整，以模型广场与充值页实时展示为准。
</Note>

## 控制成本建议

- 开发阶段使用 **mini / flash** 等小模型
- 设置合理的 `max_tokens` 上限
- 对批量任务加速率限制，避免 [429](/errors/429)
- 长文档场景先摘要再问答，减少输入 token

## 定价逻辑（简要）

- **低价档**：共享上游、智能路由，性价比高，但可能限流或偶发不稳定。
- **原价档**：按厂商官方美元标价计费，经平台默认汇率（当前 **1 USD = 7.3 CNY**）折算扣费，通常走更稳定的直连或专用线路。
- 平台采用 **薄利多销**，不靠假模型或单次暴利；不同档位差异主要在 **线路质量与限流策略**，而非模型真伪。

产品向说明（为何便宜、是否官方模型、如何选型等）见 [常见问题 · 产品与定价](/faq/common#产品与定价)。
