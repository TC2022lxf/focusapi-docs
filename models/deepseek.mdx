DeepSeek 模型通常通过 **OpenAI 兼容** 的 Chat Completions 接口调用。

## 示例

```bash
curl https://api.FocusAPI.com/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-chat",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

将 `model` 替换为模型广场中的 DeepSeek 完整 ID（如 `deepseek-reasoner` 等）。

## 使用 OpenAI SDK

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.FocusAPI.com/v1",
    api_key="sk-your-api-key",
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Hello"}],
)
```

## 说明

- **推理 / Reasoning 模型** 可能响应更慢，超时请查阅 [524](/errors/524)。
- 计费按模型倍率与 token 用量计算，见 [计费说明](/billing/pricing)。
