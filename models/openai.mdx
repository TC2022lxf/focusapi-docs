FocusAPI 对 **OpenAI Chat Completions / Responses** 等路径提供兼容转发。多数场景下，只需使用 OpenAI SDK 并修改 `base_url` 即可。

## 推荐接口

| 用途 | 方法 | 路径 |
| --- | --- | --- |
| 对话（最常用） | `POST` | `/v1/chat/completions` |
| 模型列表 | `GET` | `/v1/models` |

完整示例见 [快速开始](/quickstart) 与 [对话 API](/api/chat)。

## 模型名

请求体中的 `model` 必须与控制台 **模型广场** 中展示的 ID 一致，例如：

- `gpt-4o`
- `gpt-4o-mini`
- 其他平台已上架的 OpenAI 系列模型

<Warning>
  模型名区分大小写。复制模型广场中的完整 ID，不要自行拼接版本后缀。
</Warning>

## 认证头

```http
Authorization: Bearer sk-your-api-key
Content-Type: application/json
```

## 与原生 OpenAI 的差异（须知）

| 项目 | 说明 |
| --- | --- |
| Base URL | 使用 `https://api.FocusAPI.com/v1`，而非 `api.openai.com` |
| 计费 | 按 FocusAPI 额度与模型倍率计费，见 [计费说明](/billing/pricing) |
| 功能开关 | 部分新特性是否可用取决于平台上架情况，以控制台为准 |
