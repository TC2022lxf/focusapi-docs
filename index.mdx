## 推荐路径（新用户）

<Steps>
  <Step title="注册并登录控制台">
    在 [FocusAPI 控制台](https://focusapi.com) 完成注册。
  </Step>
  <Step title="创建 API Key">
    在控制台 **令牌** 页面创建密钥，格式通常为 `sk-` 开头。请妥善保存，密钥只显示一次。
  </Step>
  <Step title="完成第一次请求">
    跟随 [快速开始](/quickstart) 使用 `https://api.FocusAPI.com/v1` 调用 `chat/completions`。
  </Step>
  <Step title="按需查阅专题">
    流式输出、图像、音频、计费说明见对应章节。
  </Step>
</Steps>

## 核心概念

| 术语 | 说明 |
| --- | --- |
| **API Key（令牌）** | 调用 API 的凭证，放在请求头 `Authorization: Bearer sk-xxx` |
| **Base URL** | 将 OpenAI SDK 的 `base_url` 指向 `https://api.FocusAPI.com/v1` |
| **额度** | 账户可用余额，不足时可能返回与余额相关的错误 |
| **模型名** | 请求体中的 `model` 字段，需与控制台「模型广场」中可用模型一致 |
