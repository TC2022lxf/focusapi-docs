FocusAPI 是面向开发者的 **AI 模型 API 聚合平台**。本文档以“模型能力”为主线：先选择你要完成的任务，再查看可用模型、调用方式和特殊参数。

多数文本模型兼容 OpenAI SDK；通常只需要把 Base URL 改为：

```text
https://api.focusapi.cn/v1
```

## 推荐路径（新用户）

<Steps>
  <Step title="注册并登录控制台">
    在 [FocusAPI 控制台](https://focusapi.cn) 完成注册。
  </Step>
  <Step title="创建 API Key">
    在控制台 **令牌** 页面创建密钥，格式通常为 `sk-` 开头。请妥善保存，密钥只显示一次。
  </Step>
  <Step title="完成第一次请求">
    跟随 [快速开始](/quickstart) 使用 `https://api.focusapi.cn/v1` 调用文本对话模型。
  </Step>
  <Step title="按能力继续接入">
    文本、图像、视频、语音、Embedding 等能力见 [模型选择指南](/models/selection)。
  </Step>
</Steps>
